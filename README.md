# TimeSheetPro

Automated timesheet generation for Klient (Salesforce) using local Ollama AI and Windows activity monitoring.

## Repository Structure
- **/server**: Node.js Express API and SQLite database.
- **/client**: C# WPF Windows Client (Floating Widget & Activity Watcher).
- **/docs**: Project strategy, backlog, and architecture details.

## Quick Start
1. **Server**: Navigate to `/server`, run `npm install` and `npm start`.
2. **Client**: Open `/client` in Visual Studio 2022+, build and run.

See [docs/GEMINI.md](docs/GEMINI.md) for the full project vision.
